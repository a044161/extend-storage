# 增强型Storage

## 终端类型

- PC Web端
- 移动 Web端

## 功能

- expires机制
- 内容过滤
- 数据加密、解密

## 功能点

1. 提供配置选项（设置超时时间，数据加密，获取当前存储值，获取最大存储值）
2. 在单独调用的时候可进行单独配置
3. 提供清除所有数据的方法
4. 提供清除所有超时数据的方法
5. 浏览器兼容性判断
6. 添加数据
8. 更新数据超时时间
9. 更新数据值
10. 删除数据
11. 删除超时数据
12. 删除所有数据
13. 获取数据（获取超时数据时，返回的为超时提示并移除该数据）
14. 获取所有数据（获取超时数据时，返回的为超时提示并移除该数据）

## 存储格式

参照[web-storage-cache](https://github.com/WQTeam/web-storage-cache)的存储方式

```
{'key', {'c': 123, 'e': 123, 'v': 'value'}}
```

1. `'c'`：该数据创建的时间
2. `'e'`：该数据超时的时间
3. `'v'`：该数据具体的值
